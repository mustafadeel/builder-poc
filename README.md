# builder-poc

Builder.io agent skills for Auth0 authentication and React integration.

## Skills

### auth0-quickstart

Sets up Auth0 authentication in a React SPA using `@auth0/auth0-react`. Covers Auth0Provider configuration, login/logout, user profile, protected routes, and authenticated API calls.

### react-builder

Integrates Builder.io's visual headless CMS with React/Next.js projects. Covers SDK setup, custom component registration, content fetching, catch-all routing, and the visual editor.

## Installation

Copy the `.builder/skills/` directory into your project, or install individual skills:

```bash
cp -r .builder/skills/auth0-quickstart /your-project/.builder/skills/
cp -r .builder/skills/react-builder /your-project/.builder/skills/
```
